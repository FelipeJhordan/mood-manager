# mood-manager
## Descrição
Projeto feito em grupo,  sobre um aplicativo mobile para registrar emoções e mostrar gráficos/relatórios sobre seus sentimentos.

## Tecnologias Utilizadas
### Mobile
- Android
- Kotlin
- Gradle
### Back-end
- Spring
- Kotlin
- JPA
- MYSQL
- Gradle

### Infraestrutura
- EC2 ( vpn amazon )
- RDS ( banco amazon )
